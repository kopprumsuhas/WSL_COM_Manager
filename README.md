# WSL COM Manager

WSL COM Manager is a Windows GUI utility that simplifies connecting USB serial (COM) devices to WSL 2 using usbipd.

## Features
- GUI-based COM port selection
- One-click Attach / Detach to WSL
- Automatic usbipd busid detection
- Built-in logging & debug mode
- Standalone Windows installer
